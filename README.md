- What do computer scientists do and why does it matter?
Computer Scientists study, create and program computer features and applications to further computer research and create computer services for the field. This is done through algorithm creation, information computation and logic/coding crafting through computer hardware and associated software.
This helps to improve computer systems for the whole world, every software and hardware improvement from processor revolutions to the software medical professionals use to treat patients to the voice communication services used in every day life for remote workers or teachers to students, was contributed by computer
scientists. The machine that has taken over the world, computer scientists are the ones that contributed most to it's world takeover.

- How do I approach a problem as a computer scientist?
For a computer scientist, every problem is a puzzle to put together. Software and system design is built piece by piece, 1 function, method, object at a time until the full puzzle is pieced together. Problems are the same puzzle, break the problem to manageable, bit size pieces, and tackle each piece
slowly but surely putting the whole puzzle together until the problem is solved.

- What are my ethical responsibilities to the end user and the organization?
To create safe, problem free, functional systems, programs and software, that will no compromise the user that utilizes it. That is, in my own words, the important rule of program creation. End users typically do not know every end of the system they use, and put their trust in the designers, programmers, and technicians that create and maintain that program. So it is imperative we, within the best of our abilities, ensure the program works as intended, and does not endanger the user when utilized.
